如何使用GitHub托管自己的网站？

1、登录GitHub（假设有账号）

2、现在我们进入了主页面。点击右上角的加号，选择New repository（新建仓库）

3、弹出如下所示的界面，接下来在Repository name框内填写仓库的名称，我们这里以“website”为例。Description是对此仓库的描述，可以不填。

注意：仓库的公开性应选择Public也就是公开的，不然无法创建网站（当然你要是开了每月7美元的会员的话，就当我没说。）

最后面的三个选项可以暂不设置，这里我们不选

点击绿色的按钮，保存

创建仓库和index.html文件

4、成功后，他提示让你安装Github客户端，我们这里以不安装客户端的为例（因为我懒）仔细观察，Set up in Desktop下面有一行小字，点下面“creating a new file”（即：创建新文件）或者uploading an existing file（即：上传新文件，要是你本地有准备好的index.html文件点这个）

5、在小框里填上index.html下面大文本框就是写代码的地方了

6、好了之后，点击Commit new file（注：这个在页面的底部）

部署网站

7、点击Setting，往下滑，可以看到 GitHub Pages 看到Source，我们要在这开启托管功能。点击none下拉框，选择main再点击save保存

可以通过https://username.github.io/hub访问（其中，username指的是你的github用户名，hub指你的仓库的名字）

如何绑定自己的域名？ 很简单，在github pages栏中可以看到Custom domain在下面的框内填上要绑定的域名，再点击save保存即可

登录域名控制台，添加一条记录，CNAME到username.github.io保存（其中，username替换成你的GitHub用户名）

完成！
